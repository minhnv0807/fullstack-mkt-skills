# v2.5.0 — Global Marketing Cluster (2026-MM-DD)

## TL;DR (30 seconds, English)

**v2.5.0** ships the **Global Marketing Cluster** with **30 new global skills (00-29 + product-marketing-context-global)** under `skills/global/`, **8 region-aware workflows**, **5 universal agents** (auto-detect VN vs Global context), and **1 dropshipping flagship**. **Zero breaking changes** — Vietnam cluster (skills 00-21 plus personal-brand 22-28) keeps working as-is.

Pick one of 4 region variants when you start: **US** (USD/FTC), **EU** (EUR-GBP/GDPR), **SEA** (USD-or-local/PDPA per country), **LATAM** (USD-or-local/LGPD). The variant tunes currency, primary platforms, regulations, and persona framework for every downstream skill.

## TL;DR (30 giay, tieng Viet)

**v2.5.0** them cum **Global Marketing Cluster** voi **30 skills moi (00-29 + product-marketing-context-global)** trong thu muc `skills/global/`, **8 workflow co region awareness**, **5 agent universal** (tu nhan dien VN vs Global), va **1 dropshipping flagship**. **Zero breaking changes** — cum Vietnam (skills 00-21 + personal-brand 22-28) van hoat dong y nhu cu.

Chon 1 trong 4 region variant khi bat dau: **US** (USD/FTC), **EU** (EUR-GBP/GDPR), **SEA** (USD hoac local currency/PDPA tung quoc gia), **LATAM** (USD hoac local currency/LGPD). Variant se tu dong cau hinh currency, nen tang chinh, quy dinh phap luat, va persona framework cho moi skill phia sau.

## Bilingual FAQs / FAQ song ngu

**Q1 (EN): Do I need to migrate from v2.4.0?**
**Q1 (VN): Co can migration tu v2.4.0 khong?**
EN: No. v2.5.0 is purely additive. Your VN skills, context files, workflows continue working unchanged.
VN: Khong. v2.5.0 chi them, khong sua. Skills VN, context file, workflows van hoat dong y nhu cu.

**Q2 (EN): Can I use both VN and Global clusters?**
**Q2 (VN): Co the dung ca 2 cum VN va Global khong?**
EN: Yes. Many users do — VN cluster for domestic clients, Global cluster for international expansion. Agents auto-detect which file to read based on active skill.
VN: Co. Nhieu user dung ca hai — cum VN cho client trong nuoc, cum Global cho mo rong quoc te. Agents tu detect file theo skill dang active.

**Q3 (EN): Which region should I pick first?**
**Q3 (VN): Nen chon region nao truoc?**
EN: Pick the region with your highest current revenue. If unclear, default to US (highest revenue ceiling, easiest infra: Stripe, Shopify, Klaviyo). See `docs/global-region-guide.md` for decision tree.
VN: Chon region co revenue cao nhat hien tai. Neu khong ro, default US (revenue ceiling cao nhat, infra de: Stripe, Shopify, Klaviyo). Xem `docs/global-region-guide.md` cho decision tree.

**Q4 (EN): When will Japan/Korea/Australia be added?**
**Q4 (VN): Khi nao co Japan/Korea/Australia?**
EN: APAC is roadmapped for v2.6.0 if criteria are met (5+ issues, 1000 stars, native reviewers). See `docs/apac-roadmap.md`.
VN: APAC nam trong roadmap v2.6.0 neu dat criteria (5+ issue, 1000 star, native reviewer). Xem `docs/apac-roadmap.md`.

**Q5 (EN): Is dropshipping still viable in 2026?**
**Q5 (VN): Dropshipping con kha thi nam 2026 khong?**
EN: Yes, but harder than 2020-2022 era. 90% fail rate; survivors treat it as 12-month learning investment, not 30-day get-rich. Read `docs/dropshipping-guide.md` Chapter 1 for reality check.
VN: Co, nhung kho hon thoi 2020-2022. 90% fail; survivors coi day la dau tu hoc 12 thang, khong phai get-rich 30 ngay. Doc `docs/dropshipping-guide.md` Chuong 1 de reality check.

**Q6 (EN): What if my customers are in multiple regions?**
**Q6 (VN): Khach hang o nhieu region thi sao?**
EN: Run `product-marketing-context-global` once per region; rename output files (e.g. `product-marketing-context-global-us.md`, `-eu.md`). v2.5.0 supports this manually; v2.6.0 may add an explicit flag.
VN: Chay `product-marketing-context-global` 1 lan moi region; doi ten file output (vi du `product-marketing-context-global-us.md`, `-eu.md`). v2.5.0 support manually; v2.6.0 co the them flag.

**Q7 (EN): Does the global cluster support languages other than English?**
**Q7 (VN): Cum global co support ngon ngu khac ngoai English khong?**
EN: Yes. Region variants set the language: EU multi-language (de/fr/es/it/nl/pl), SEA (id/th/tl/ms/zh-SG), LATAM (es/pt-BR). Skills 04 (video), 05 (ad copy), 14 (email) write in target language.
VN: Co. Region variants tu dong set ngon ngu: EU da ngon ngu (de/fr/es/it/nl/pl), SEA (id/th/tl/ms/zh-SG), LATAM (es/pt-BR). Skills 04 (video), 05 (ad copy), 14 (email) viet ngon ngu target.

**Q8 (EN): Can I customize the region variants?**
**Q8 (VN): Co the customize region variants khong?**
EN: Yes. The 12-section foundation file is your customization layer. Edit `.agents/product-marketing-context-global.md` directly to adjust personas, competitors, voice, proof points.
VN: Co. File foundation 12-section la layer customize. Sua truc tiep `.agents/product-marketing-context-global.md` de chinh personas, competitors, voice, proof points.

**Q9 (EN): How do I report a region-specific bug?**
**Q9 (VN): Bao bug specific cua region nhu the nao?**
EN: Open a GitHub issue with label `region:US`, `region:EU`, `region:SEA`, or `region:LATAM`. Include the skill name, the wrong output, and what you expected.
VN: Mo issue GitHub voi label `region:US`, `region:EU`, `region:SEA`, hoac `region:LATAM`. Bao gom ten skill, output sai, va output mong doi.

**Q10 (EN): Will the VN cluster keep getting updates?**
**Q10 (VN): Cum VN co tiep tuc duoc update khong?**
EN: Yes. VN cluster is the project's primary cluster (founder + initial users are VN-based). Global cluster is parallel, not a replacement.
