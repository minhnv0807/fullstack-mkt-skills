## 10 FAQs (chon tu `docs/faq.md` / selected from full FAQ)

**Q1: Personal brand co khac product marketing khong? / Is personal brand different from product marketing?**

VN: Co. Product = ban san pham. Personal brand = ban xay than the (chinh ban). Co the dung CA HAI cung luc.

EN: Yes. Product = sell products. Personal brand = build yourself as the product. Both can be used together.

**Q2: Toi can bao nhieu thoi gian moi tuan? / How much time do I need per week?**

VN: Toi thieu 4-6 gio/tuan cho content. AI avatar giam xuong 2-3 gio/tuan sau khi setup.

EN: Minimum 4-6 hours/week for content. AI avatar reduces to 2-3 hours/week after setup.

**Q3: Co bat buoc disclose AI avatar khong? / Is disclosing AI avatar mandatory?**

VN: O VN, theo Nghi dinh 147/2024, co. Xem `references/ai-video-disclosure-vn.md` cho 5 template disclosure.

EN: In VN per Decree 147/2024, yes. See `references/ai-video-disclosure-vn.md` for 5 disclosure templates.

**Q4: Tool nao toi nen dung? HeyGen hay Synthesia? / Which tool — HeyGen or Synthesia?**

VN: Founder/coach -> HeyGen (lifelike). Creator -> Synthesia (so luong + workflow). Xem reference deep-dive.

EN: Founder/coach -> HeyGen (lifelike). Creator -> Synthesia (volume + workflow). See deep-dive reference.

**Q5: Ngan sach toi thieu / Minimum budget?**

VN: Tier 1 (free): $0 — basic avatar tools. Tier 2: ~$50/thang. Tier 3 (pro): $200-500/thang.

EN: Tier 1 (free): $0 — basic avatar tools. Tier 2: ~$50/month. Tier 3 (pro): $200-500/month.

**Q6: Toi viet tieng Anh hay tieng Viet? / Should I write in Vietnamese or English?**

VN: Audience VN -> tieng Viet. Global audience -> tieng Anh. Skill 26 ho tro ca hai.

EN: VN audience -> Vietnamese. Global audience -> English. Skill 26 supports both.

**Q7: Khi nao monetize? / When to monetize?**

VN: Sau 6 thang co audience > 5K engaged followers. Skill 27 co 3 funnel khac nhau theo stage.

EN: After 6 months with > 5K engaged followers. Skill 27 has 3 funnel versions by stage.

**Q8: Community vs audience khac gi? / Community vs audience difference?**

VN: Audience = nguoi xem 1-chieu. Community = nguoi tuong tac voi nhau (Zalo group, Skool). Skill 28.

EN: Audience = one-way viewers. Community = members interact with each other (Zalo, Skool). Skill 28.

**Q9: Toi co the chuyen variant sau khi chay khong? / Can I switch variants later?**

VN: Co. Chay lai skill 22 voi variant khac. File cu se bi ghi de neu xac nhan.

EN: Yes. Re-run skill 22 with different variant. Old file is overwritten if confirmed.

**Q10: Voice clone co rui ro phap ly khong? / Are there legal risks with voice cloning?**

VN: Khong neu clone GIONG CUA BAN. Khong dung voice nguoi khac neu khong co consent. Xem `references/voice-clone-prompts-vn.md`.

EN: No risk if cloning YOUR OWN voice. Never use others' voices without consent. See voice-clone-prompts-vn.md.
