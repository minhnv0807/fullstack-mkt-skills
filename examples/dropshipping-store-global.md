## 4. Output from skill 17-pricing-strategy-global (Dropshipping markup)

**File:** `pricing-strategy-cool-pet-bed-20260508.md`

### Pricing math
- **Cost:** $8 product + $5 shipping (CJ US) = $13 landed cost per unit
- **Sell price:** $39.99
- **Profit per unit (before ad cost):** $26.99
- **Markup:** 3.07x (within 3-5x sweet spot for dropshipping)
- **Break-even ROAS:** $39.99 / $26.99 = 1.48 (need ROAS > 1.48 to be profitable)
- **Target ROAS:** 2.5-3x for healthy 30-40% net margin
- **Payment processing fee:** 2.9% + $0.30 = $1.46 per order → adjust profit to $25.53

### Price ladder (encourage upsell — increases AOV)
- 1 unit: $39.99
- 2 units: $69.99 (save $9.99 → "BEST FOR 2 PETS")
- 3 units + free toy: $99.99 (save $19.97 + free $9.99 toy → "BEST VALUE")

### AOV impact projection
| Scenario | Bundle mix | AOV |
|----------|-----------|-----|
| All 1-unit | 100% × $39.99 | $39.99 |
| Realistic mix | 70% × $39.99 + 25% × $69.99 + 5% × $99.99 | $50.49 |
| Optimistic mix | 50% × $39.99 + 35% × $69.99 + 15% × $99.99 | $59.24 |

### Discount & free shipping strategy
- Free shipping baked into price (NOT shown as discount — psychological anchor)
- 10% off first order via popup (email capture trigger)
- "27 left in stock" urgency (real-time inventory ticker app)
- Cart abandonment: 10% off code via email if cart abandoned >1 hour

### Top 3 issues identified

**Issue 1: Targeting too broad (CRITICAL)**
- Current: "Dog owners US 25-65" interest stack
- Problem: 30% of clicks from non-buyers (under-25, low-AOV regions)
- Fix: Narrow to "Dog owners + Premium pet brands (Chewy, BarkBox)" + age 28-55

**Issue 2: Hero LP missing video (HIGH)**
- Current: static hero image only
- Problem: bounce rate 68% on mobile (vs 45% benchmark)
- Fix: Add 8-second auto-play hero video (dog walking onto bed, falling asleep)

**Issue 3: Bundle upsell underperforming (MEDIUM)**
- Current: bundle option below the fold, AOV stuck at $42
- Problem: only 12% of orders are bundles
- Fix: Move bundle selector ABOVE add-to-cart button + add "save $9.99" sticker

### Quick Wins (do this week)
1. Pause 3 lowest-performing ad sets (saving ~$200/week wasted spend)
2. Launch new UGC creative (creator already booked, $300 cost)
3. A/B test bundle position on LP (Convert.com test)
4. Retarget cart abandoners with 10% off code (Klaviyo flow)

### Projected impact if Quick Wins applied
